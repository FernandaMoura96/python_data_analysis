# 🐍 Python para Análise de Dados 

Scripts desenvolvidos durante o curso **Teo Me Why**, com foco em análise de dados com Python.

## 📦 Conteúdo

- Manipulação e limpeza de dados com **Pandas**
- Operações numéricas com **NumPy**
- Análise exploratória de dados (EDA)



## 🛠️ Tecnologias

- Python 3.x
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📚 Sobre o curso

O [Teo Me Why](https://www.twitch.tv/teomewhy) é um canal de ensino de dados e tecnologia, com aulas ao vivo e conteúdo gratuito focado em análise de dados, SQL, Python e muito mais.
